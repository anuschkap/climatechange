Gamlss regressions all datasets - age

Anuschka Peelen

2024-01-12

In this script I use loops to perform gamlss regressions for all 52 attitudes. The lists that I will get as output are saved with all the information, after which - in another script - i will extract the raw effects sizes used as input for the meta-regressions. In this specific script, I will only include the effects of age to test the idea of opposing effects of gender + education versus age. Furthermore, I will only run the independent variables model, as to test the theoretical idea underlying this test, the interaction model is less important.

ess_complete <- esstotal %>% drop_na()

#Estimate a second model with the indicators + time

deps <- c("worry", "cause", "pers_resp")

ess_list_preds <- list()

# Loop through each dependent variable
  for (i in 1:length(deps)) {
  # Specify the sigma formula
  sigma.formula <- as.formula(paste("~", paste(predictors, collapse = " + ")))
  # Perform the linear regression
  ess_fit <- gamlss(as.formula(paste(deps[i], "~", paste(predictors, collapse = " + "))), sigma.formula = sigma.formula, family=NO(mu.link = "identity", sigma.link = "identity"), data = ess_complete, weights = weightvec)
  
  ess_list_preds[[i]] <- ess_fit
}
